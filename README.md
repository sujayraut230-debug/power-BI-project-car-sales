# power-BI-project-car-sales

Car Sales Dashboard Analysis
Overview

Developed an interactive Power BI dashboard to analyze car sales performance, customer trends, regional sales distribution, and pricing insights.

Features
KPI Cards for Average Car Price, Average Cost Price, and Discount
Region-wise Sales Analysis
Customer and Country Filters
Maximum Price Analysis by Car Make
Interactive Slicers and Drill-down Reports
Tools Used
Power BI
DAX
Excel/CSV
Data Visualization
Key Insights
Identified highest-priced car brands.
Analyzed sales distribution across regions.
Compared discounts and pricing trends.
Improved decision-making through interactive dashboards.
